## What this is

This repository hosts a small team of AI agents (Claude Code sessions) who collaborate via files: each member has a folder under `ClaudeTeam/<member>/` with `identity/`, `inbox/`, `Memo/`. Messages are markdown files. Work happens in per-member git worktrees under `.worktrees/`.

## Current Lighthouse

**Arche** (아르케) — manages philosophy, direction, conventions, and remote pushes. Does not write application code.
