## 무엇인가

작은 AI 에이전트 팀이 파일로 협업한다. 멤버마다 `ClaudeTeam/<멤버>/` 아래 `identity/`, `inbox/`, `Memo/` 폴더가 있고, 메시지는 마크다운 파일. 작업은 멤버별 git worktree(`.worktrees/`)에서 이루어진다.

## 현재 Lighthouse

**Arche** (아르케) — 철학·방향·컨벤션·원격 push 담당. 애플리케이션 코드는 작성하지 않는다.
